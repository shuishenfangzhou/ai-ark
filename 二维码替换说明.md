# 二维码替换说明

## 当前二维码位置

### 1. 登录模态框中的二维码

**文件路径**: `D:\AI工具箱\index.html`

**代码位置**: 第 287-288 行

**当前代码**:
```html
<div class="relative mx-auto w-56 h-56 bg-white border-2 border-slate-100 rounded-xl flex items-center justify-center mb-6 group cursor-pointer shadow-inner">
    <i class="fa-solid fa-qrcode text-8xl text-slate-800"></i>
    <!-- 这是一个FontAwesome图标，不是真实图片 -->
</div>
```

**如何替换为真实二维码图片**:

#### 方案A：替换为图片文件（推荐）

1. 准备你的微信公众号二维码图片（建议尺寸：400x400像素）
2. 将图片保存到项目目录，例如：`public/wechat-qr.png`
3. 修改代码：

```html
<div class="relative mx-auto w-56 h-56 bg-white border-2 border-slate-100 rounded-xl flex items-center justify-center mb-6 group cursor-pointer shadow-inner">
    <img src="/wechat-qr.png" alt="微信公众号二维码" class="w-full h-full object-contain">
    <div id="qr-overlay" class="absolute inset-0 bg-white/95 flex flex-col items-center justify-center opacity-0 group-hover:opacity-100 transition-all duration-300 backdrop-blur-sm">
        <div class="w-12 h-12 bg-green-500 rounded-full flex items-center justify-center mb-3 shadow-lg transform scale-0 group-hover:scale-100 transition-transform duration-300 delay-75">
            <i class="fa-solid fa-check text-white text-xl"></i>
        </div>
        <span class="text-sm font-bold text-slate-800">点击模拟关注</span>
    </div>
</div>
```

#### 方案B：使用在线图片URL

如果你有二维码的在线URL，可以直接替换：

```html
<div class="relative mx-auto w-56 h-56 bg-white border-2 border-slate-100 rounded-xl flex items-center justify-center mb-6 group cursor-pointer shadow-inner">
    <img src="https://your-cdn.com/wechat-qr.png" alt="微信公众号二维码" class="w-full h-full object-contain">
    <!-- 悬浮层保持不变 -->
</div>
```

## 替换步骤总结

1. **准备二维码图片**
   - 登录微信公众平台
   - 下载公众号二维码（带参数或不带参数）
   - 建议尺寸：400x400像素
   - 格式：PNG或JPG

2. **放置图片文件**
   ```
   D:\AI工具箱\public\wechat-qr.png
   ```

3. **修改代码**
   - 打开 `D:\AI工具箱\index.html`
   - 找到第 287-288 行
   - 将 `<i class="fa-solid fa-qrcode...">` 替换为 `<img src="/wechat-qr.png">`

4. **重新部署**
   ```bash
   npm run build
   vercel --prod
   ```

## 注意事项

- 二维码图片路径使用 `/wechat-qr.png`（以/开头表示从根目录开始）
- 确保图片文件名和代码中的引用一致
- 如果二维码加载失败，可以添加onerror处理：
  ```html
  <img src="/wechat-qr.png" onerror="this.src='/fallback-qr.png'" alt="二维码">
  ```
